Responsive Web Design

Responsive Web designs, or RWDs, respond to their environment based on the screen size being used. They make use of flexible layouts that allow for a more accessible and readable design.




Indepth

Websites should always try to maintain a responsive and accessible design, as it helps encourage web traffic and happy viewers.
Without RWD sites could easily run amiss with messy displays and improper navigation, leading to one big headache for everyone involved.
